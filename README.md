# Student Marks Manager

A terminal-based Student Marks Manager built using JavaScript and Node.js.  
This project allows users to manage student records interactively through the command line.

## Features

- Add new students
- View all students
- Search student by name
- Calculate average marks
- Find topper
- Pass/Fail filtering
- Interactive menu system
- Input validation

## Technologies Used

- JavaScript
- Node.js
- Readline Module

## Installation

Clone the repository:

```bash
git clone <your-repository-link>
```

Move into the project folder:

```bash
cd student-marks-manager
```

Run the project:

```bash
node app.js
```
